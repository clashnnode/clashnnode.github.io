FAQ

Hysteria2 的多路复用对短连接有何优势？
多路复用允许多个短连接共享同一通道，减少握手延迟和资源消耗。适合高频短连接应用，如游戏和即时通讯，可提高吞吐量和连接稳定性，优化网络体验。
Trojan-Go 的 MUX 功能能降低延迟吗？
启用 MUX 后，多条 TCP/QUIC 连接可以共享单个隧道，减少握手次数和连接延迟。对于频繁短连接应用如游戏或社交软件，可以明显提升响应速度。
Nekoray 如何实现节点订阅自动更新？
支持订阅 URL 自动更新节点，程序定期下载并替换旧节点。保证节点列表最新，减少手动维护，提高使用效率和便利性，同时保证网络访问稳定性。
V2Ray 如何设置多入口实现不同访问控制？
通过定义多个 inbound，对不同端口绑定不同的 tag 并在路由中匹配流量。可针对内网用户、远程用户或特定 IP 段应用不同规则，实现灵活访问控制与流量隔离。
V2Ray 如何使用 WebSocket 传输？
在 V2Ray 配置文件中将传输方式设置为 ws，即启用 WebSocket 模式。结合 CDN 和 TLS，可实现安全高速的代理链路。WebSocket 常用于伪装成 HTTPS 流量，从而降低被识别风险。
VMess AEAD 加密如何确保数据安全？
AEAD 加密防止数据篡改、重放攻击和流量特征泄露。在高封锁环境下，保证节点可用，提升数据安全和连接稳定性，降低被检测风险，确保长期安全访问。
VMess AEAD 加密在高封锁环境的优势？
AEAD 加密可防止数据篡改和流量特征泄露，降低被检测风险。在高封锁网络中，保证节点可用性和连接稳定性，同时提升数据传输安全性，确保长期安全访问。
SSR 的协议参数和混淆参数如何配置最安全？
选择 AEAD 或协议插件可提升安全性，混淆方式如 tls1.2_ticket_auth 可以模拟正常 HTTPS 流量。合理搭配协议和混淆参数，能够有效抵御 DPI 检测和流量封锁。
Matsuri 节点延迟排序功能如何使用？
Matsuri 会自动测试节点延迟，并根据结果排序。用户可以选择延迟最低的节点使用，从而优化访问速度和稳定性，尤其适合游戏、视频和海外网站加速。
Mellow TUN 模式对游戏流量有何作用？
TUN 模式拦截系统 IP 层流量，实现全局代理。游戏流量通过低延迟节点传输，减少丢包和延迟，提高游戏体验的流畅性和网络稳定性。
